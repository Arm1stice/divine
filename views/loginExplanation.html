Login Explanation

Why do you need my Steam login credentials?

Divine aims to provide detailed statistics on your match history, and obtains information on matches in multiple ways. First, it attempts to find new matches using the Steam Web API (using the API key you provided), then if it finds a new match, then it will grab match information from the Web API, and grab MMR data by logging into the game and grabbing current data. NO information is shared with any outside server, and the program does not store your password.
